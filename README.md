# Study Night Production Project

This project was completed as part of my Front End Web Developer Nanodegree.

The goal of this project was to test, optimize, and prepare a front-end application for production. I worked with modern tools to improve workflow, automate tasks, and make sure the app was ready for deployment.

## Features

- Unit testing with Mocha and Chai
- End-to-end testing with Cypress
- Build process using Gulp
- Bundling and optimization with Parcel

## Tools Used

- JavaScript
- Mocha
- Chai
- Cypress
- Gulp
- Parcel
- npm

## What I Learned

This project helped me understand how front-end apps are prepared for production. I practiced writing tests, using build tools, managing dependencies, and improving project structure.
